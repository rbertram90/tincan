# Tin Can Forum

Tin Can is a lightweight web forum written in PHP.

## Requirements

* PHP
* MySQL

## Configuration

Copy tc-config-example.php to tc-config.php and change these values to suit your environment.

* `TC_BASE_PATH` = `/var/www/YOUR_SITE`
* `TC_DB_HOST` = `http://localhost`
* `TC_DB_USER` = `root`
* `TC_DB_PASS` = `root`
* `TC_DB_NAME` = `tincan`
